How It Works

Three Steps.
That's It.

01

Browse LinkedIn

Install the extension and head to your LinkedIn feed. A 📸 Snap button automatically appears on every post.

02

Click Snap

Click the Snap button. A beautiful customization dashboard opens with a live preview of your post card.

03

Customize & Export

Choose your theme, background, and adjust the sliders. Hit Download PNG or Copy to Clipboard. Done.

Get Started

Install in
60 Seconds.

1

Clone the Repository

Run git clone https://github.com/HanshalBobate/LinkSnap.git or download the ZIP from GitHub.

2

Open Chrome Extensions

Navigate to chrome://extensions in your Chrome browser.

3

Enable Developer Mode

Toggle "Developer mode" in the top-right corner of the extensions page.

4

Load Unpacked

Click "Load unpacked" and select the LinkSnap/ folder. That's it!
